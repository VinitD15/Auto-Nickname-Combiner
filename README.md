# Auto Nickname Combiner
This is a fun Python console application that creates creative nicknames by combining parts of two names.  
It helps users explore string manipulation, randomness, and user input while generating multiple nickname suggestions.

## Features
- Combine two names in different styles to create nicknames  
- Multiple combination options:
  - First 2 letters of Name1 + last 2 letters of Name2  
  - First half of Name1 + second half of Name2  
  - Random characters from both names  
  - First 3 letters of both names  
  - Reversed combinations  
- Generate fun nickname meanings  
- Keep a session history of all nicknames generated  
- Optional colored output using `colorama`

## Tools and Libraries
- Python 3.9.6 +
- Core libraries: `random`, `time`  
- Optional: `colorama` for colored terminal text  

## How to Use
1. Clone or download the repository  
2. Open the project in VS Code  
3. (Optional) Create and activate a virtual environment  
4. Install optional library:
   ```bash
   pip install colorama
5. Run the program:
    python nickname_combiner.py

## Concepts Used
-- Python strings and slicing
-- Random module for nickname variations
-- Loops and conditional statements
-- Functions for modular design
-- Lists to store session history
